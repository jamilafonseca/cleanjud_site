01) Sobre o Sistema XML VALID


O sistema tem o objetivo de validar o preenchimento e a consistência das informações do arquivo XML (Metadados dos Processos Jurídicos),
ANTES antes do envio das informações ao CNJ DATAJUD.

O XML Valid faz a identificação dos problemas, gerando um relatório automático das inconsistências e corrigindo, na medida do possível, as informações faltantes ou inconsistentes.

02) Compatibilidade

Compatível com o Sistema Operacional Windows Desktop 7+ (ou posterior)
ou Windows Server.


03) Download do Sistema:


O sistema é instalado na máquina (ou servidor) local do Tribunal,
de onde está armazenado o arquivo XML original.

A instalação é simples. É apenas um (1) executável.

<< Clique no botão ao lado para iniciar o Download.


04) Instalação


Após o download, para instalar, basta descompactar o arquivo ZIP na pasta destino.



05) Instrução de Uso


Para executar o programa:

1) Entre no Modo de Linha de Comando (MS PROMPT)

2) Digite o nome do programa e o nome do arquivo a ser analisado, conforme abaixo:




>> cleandjud_xml_valid.exe exemplo.xml
